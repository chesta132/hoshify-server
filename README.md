# Project Is Discontinued

This repository is no longer maintained.

## Reason for Discontinuation
The project was originally designed using a monolithic architecture, which later proved to be unsuitable for the intended scale and structure of the backend. For the system to function properly and remain maintainable, it should have been built using a microservices-based architecture instead.

Refactoring the entire codebase to follow a proper microservice design would require starting from scratch. Because of this, development on this repository has been discontinued.
